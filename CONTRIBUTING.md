# Contributing to Garmin Connect Export

Thank you for your interest in contributing to this project! Contributions are warmly welcome.

## How to Contribute

### Reporting Issues

If the script stops working with Garmin Connect or you encounter bugs:

1. Check existing [issues](https://github.com/aj-enns/garmin-connect-export/issues) to see if it's already reported
2. If not, open a new issue with:
   - A clear description of the problem
   - Steps to reproduce the issue
   - Your Python version and OS
   - Any error messages or logs

### Suggesting Features

New features, however simple, are encouraged! Please:

1. Open an issue to discuss the feature first
2. Explain the use case and how it would benefit users
3. Be open to feedback and discussion

### Submitting Pull Requests

1. **Fork the repository** and create your branch from `master`
2. **Make your changes**:
   - Follow the existing code style
   - Keep functions focused and well-documented
   - Add comments for complex logic
3. **Test your changes** with your Garmin Connect account
4. **Update documentation** (README.md) if needed
5. **Submit a pull request** with:
   - Clear description of what changes were made
   - Reference any related issues
   - Explanation of why the change is needed

## Code Style

- Use Python 3.x syntax
- Follow PEP 8 guidelines where reasonable
- Use meaningful variable and function names
- Keep functions small and focused
- Add docstrings to functions

## Testing

- Test the script with various activity types (running, cycling, indoor, outdoor)
- Verify all export formats (GPX, TCX, FIT, JSON) work correctly
- Ensure credentials are never exposed in logs or output

## Questions?

Feel free to open an issue for any questions about contributing!
